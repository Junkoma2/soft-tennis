# soft-tennis

PCとスマホで遊べるソフトテニス（軟式テニス）のダブルスラリーゲームです。
ポジション（後衛/前衛）と陣形（雁行陣/ダブル後衛/ダブル前衛）を選び、
「ため打ち」でコースと球威を打ち分けて相手前衛との駆け引きを楽しめます。

## 遊び方

1. 開始画面でポジション（後衛を操作/前衛を操作）と陣形を選び、「試合を始める」を押す
2. 操作しない相方と相手チーム（後衛・前衛）はAIが自動で動きます
3. 自分のサーブの番では、サーブ方式（アンダーカット／オーバー）を選び、1回目の操作でトス、トスの頂点付近で2回目の操作をして打ちます。トス中も左右に移動でき、立ち位置と打つタイミングで対角サービスコート内の左/中/右を狙えます（2本制、ダブルフォルトで失点）
4. ラリー中は「ため打ち」: ボタンを押している間ためて、ボールが打点に来ると自動でスイング（途中で離してもスイング）。ためが長いほど球速が上がります。ため中に左右を入力した方向（クロス/ストレート）へ、入力しなければミドルへ飛びます
5. リングのマーカーが着地予定地点を示し、自分側に来るボールは黄色で表示されます
6. 相手前衛がネット際でポジションを取り、低いボールが近くを通るとボレーでカットしてきます。空いたコースを狙うか、ロブで頭上を越しましょう
7. ソフトテニスのカウント（0 → 1 → 2 → 3 → ゲーム）でスコアを表示。3-3はデュースとなり、2点差がつくまで続きます
8. 先に3ゲーム取った方が勝利（5ゲームマッチ）。2-2となった最終ゲームはファイナルゲーム（7ポイント先取・6-6デュース・2ポイントごとにサーブ交代）です

## サーブ順（JSTA競技規則準拠）

- サービスは1ゲームごとに両チーム交互
- 競技規則第24条により、**同じゲームの中でペアの2人が2ポイントずつかわるがわるサーブ**します（前衛もサーブを打つ）。このゲームでは「1人目=後衛、2人目=前衛」の順で固定しています
- 前衛がサーブする番は、打つまでベースライン後方に留まり、打った後にサービスダッシュで前へ詰めます
- 相方の番のサーブはAIが自動で打ちます
- ファイナルゲームは2ポイントごとに「自チーム1人目→相手1人目→自チーム2人目→相手2人目」の輪番です
- サーブ位置はポイントごとに右/左交互。レシーブは簡略化のため常に後衛役が担当します

## ショットの仕組み

- **球種**：シュート（低く速い・ドライブ回転）／ロブ（高い弧で前衛の頭上を越える）／ドロップ（ネット際に短く落とす・スライス回転）
- **回転とバウンド**：ドライブ回転はバウンド後に相手へ食い込み、スライス（カットサーブ含む）はバウンド後にかなり止まって低く滑ります
- **ため**：ためるほど球速アップ（最大+30%）。ため中は移動が遅くなります
- **コース**：ため中の左右入力で決定。立ち位置との関係でクロス/ストレートになります
- **フォア・バック**：ボールが来た側によってフォアハンド／バックハンドが決まり、バック側は威力・精度が落ちます
- **サーブ**：アンダーカット（遅いがバウンド後に止まって攻められにくい）／オーバー（速いがフォルトのリスクあり）

## 操作方法

### PC

- 矢印キー（またはWASD）：前後左右の自由移動
- スペース長押し：ため（ボールが打点に来ると自動スイング、離してもスイング）
- ため中に ← / →：コース指定（押さなければミドル）
- Shift+スペース：ロブ ／ Ctrl+スペース：ドロップ
- サーブ：スペース1回目でトス → 頂点付近でもう一度スペース

### スマホ

- 左下のスティック：前後左右の自由移動
- 右下のボタン（打つ/ロブ/ドロップ）長押し：ため → 離して打つ
- ため中はスティックの左右でコース指定
- サーブ：ボタンかコートをタップ（1回目トス → 2回目打つ）

## パラメータ調整

球速・足の速さ・ため効果・回転係数などは `app.js` 冒頭の `TUNING` オブジェクトに一元化しています。数値を書き換えるだけでゲームバランスを調整できます。
選手の能力値は `makeStats()`（power / serve / speed / reach / control / volley）に一元化しており、将来の育成要素はこの値を書き換えるだけで反映されます。
