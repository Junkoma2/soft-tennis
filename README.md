# soft-tennis

PCとスマホで遊べるソフトテニスのラリーゲームです。

## 遊び方

1. 「試合を始める」ボタンを押して試合を開始
2. 自分のラケット（下側）を左右に動かしてボールを打ち返す
3. 自分のサーブの番では、画面をタップ（PCはクリック）するとサーブを打てる
4. CPU相手とラリーを続け、ポイントを重ねる
5. 軟式テニスのポイント表記（0 → 15 → 30 → 40 → デュース → アド）でスコアを表示
6. 先に4ポイント以上かつ2点差をつけるとそのゲームを獲得
7. 3ゲーム先取で試合終了

## 操作方法

- PC: ← → キー、またはマウスを左右に動かす
- スマホ: 画面を左右にドラッグ
