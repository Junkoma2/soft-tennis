# soft-tennis

ソフトテニス（軟式テニス）のダブルスラリーゲームです。
**PC・横画面（ランドスケープ）推奨。** 横長の中継カメラ視点でコートを台形に描画し、
前衛・後衛の左右位置がひと目で読めます。キーボードで快適に操作できます
（スマホ・縦画面でも遊べますが当面は簡易対応です）。
ポジション（後衛/前衛）と陣形（雁行陣/ダブル後衛/ダブル前衛）を選び、
打点とコース選択で相手前衛との駆け引きを楽しめます。

## 遊び方

1. 開始画面でポジション（後衛を操作/前衛を操作）と陣形を選び、「試合を始める」を押す
2. 操作しない相方と相手チーム（後衛・前衛）はAIが自動で動きます
3. 自分のサーブの番では、パワー（弱/中/強）・回転（弱/中/強）を設定 → **全員（4人）の準備が整ってから**左クリックでトスを上げ、ボールが適正打点の高さ（ゲージの「適正」マーカー）に来たら**左クリック=フラット／右クリック=スライス／Space+左クリック=アンダーカット（セカンド向け安全球）／Space+右クリック=攻撃的カット（速くて鋭いがリスク高め）**で打ちます（2本制、ダブルフォルトで失点）
4. 相手のサーブは**種類（フラット/スライス/アンダーカット/攻撃的カット）が打つ前に表示**されます。サーブは対角のサービスコートにしか来ないので、レシーバーはその対角範囲の真ん中で構えます。**スライス/アンダーカット告知ならサービスライン付近まで前に出て、フラット/攻撃的カット告知なら下がって**待ちます
5. ラリー中はボールが打点ゾーンに入ると自動で「ため」が始まり、**マウスが指すコート地点**へ**着地点カーソル**（ゴーストリング）が追従して狙います。**左クリック=シュート、右クリック=カット、Spaceを押しながらクリック=ロブ**でその場でスイング（**マウスを動かさなくても安全なミドル深めへ打ちます**。スマホはスティックでカーソル移動）
6. 打点の良し悪しで球質が変わります。詰まったときに鋭い角度を狙っても浅い角度に補正され、適正打点なら狙いどおりの鋭いコースへ飛びます
7. 相手前衛はセオリーに従ってコースを守り、ポーチ（飛び出しボレー）も仕掛けてきます。前衛のいないコースを突けば相手後衛を走らせて抜けます。**両チームの前衛はレシーブが返るまでポジション移動しません**

## ゲームのルール（JSTA競技規則準拠）

- カウントは 0 → 1 → 2 → 3 → ゲーム（3-3はデュース、2点差で決着）
- 5ゲームマッチ・3ゲーム先取。2-2のファイナルゲームは7ポイント先取（6-6デュース）
- サービスは1ゲームごとに両チーム交互。同じゲームの中でペアの2人が2ポイントずつかわるがわるサーブします（前衛もサーブを打つ）。ファイナルゲームは2ポイントごとに4人の輪番
- 前衛がサーブする番は、打った後にサービスダッシュで前へ詰めます。相方の番のサーブはAIが自動で打ちます

## ショットの仕組み

### 球種（選択は3系統。内部で5種へ自動振り分け）

操作を簡単にするため、プレイヤーが選ぶ球種は **シュート / カット / ロブ** の3つです。実際の打球は、打点の高さやための強さに応じて内部で5種類の物理（軌道・回転・バウンド・色）へ自動的に振り分けられます。

| 選択（系統） | 操作 / 色 | 内部の球種と振り分け |
| --- | --- | --- |
| シュート | 左クリック / オレンジ | **打点が高い → フラット**（白・速くて低い、無回転気味）／**通常〜低い → ドライブ**（オレンジ・トップスピンで食い込む） |
| カット | 右クリック / 水色 | **着地カーソルが深い → スライス**（水色・食い込んで深い、止まって滑る）／**カーソルが浅い → ドロップ**（紫・ネット際に短く落とす）。深さは狙い（カーソル位置）で連続的に決まる（ため量では分岐しない） |
| ロブ | Space+クリック / 黄 | 意図的な高弾道で前衛の頭上を越える |
| スマッシュ | 自動 / 赤 | **ネット前で高い球を捉えると自動発動**。上から速く鋭い角度で叩き込む決め球（球種選択に関わらず判定。ロブ選択時のみ対象外） |
| フラットサーブ | 左クリック | サーブ専用。4種の中で最速・最深。トス頂点付近の高い打点で打つ |
| スライスサーブ | 右クリック | サーブ専用。フラットよりやや低い打点で打ち、フラットより遅れて落下する |
| アンダーカットサーブ | Space+左クリック | **サーブ専用**。低いトス位置から山なりに打つ、セカンドサーブ向けの安全球。確実に入りやすい |
| 攻撃的カットサーブ | Space+右クリック | **サーブ専用**。フラットとアンダーカットの中間の打点で速く鋭く切れる/伸びる攻め球。適正ゾーンが狭くリスク高め |

軌道（トレイル）と球の縁は内部の球種の色で表示されます（同じシュートでもフラット＝白／ドライブ＝オレンジに変わる）。中ロブはありません。

### 狙いは着地点カーソル（ため中に自由移動）

- ため中、マウスが指す相手コート地点へ**狙いマーカー（ゴーストリング）**が追従する（スマホはスティックで移動）。スイング時にそこへ向けて打つ。スクリーン座標→地面の逆投影でマウス位置をコート座標へ変換している
- カーソルの移動範囲はコート内（マージンあり）。**適正打点なら基本コートに収まり**、悪い打点・強打時にだけアウトのリスクが出る
- カーソル未操作でも**デフォルト位置（安全なミドル深め）**へ打つ（空振りにはならない）

### 打点システム（高さ・前後・体との距離で球質が決まる）

- **体の横の距離**: 体の横・少し前の適正打点ほど角度と球速が出る。詰まる（体に近い）ほど「狙える角度の幅」が段階的に狭くなり、**鋭い角度にカーソルを置いても浅い角度に補正される**。詰まると引っ張り方向はほぼ真っ直ぐのみになり、流し方向は比較的出しやすい
- **前後**: 前すぎる打点は引っ張りが強く低弾道で速い球に。後ろの打点は流しが強く弱い球になる
- **高さ**: 高い打点は速く低弾道（トップ打ちフラットで25m/s程度）、低い打点はすくい上げで山なりの遅い球になる
- 打点が悪いほど狙いが散らばり、ミス（アウト/ネット）が出やすくなる
- **ため**: ためが長いほど鋭い角度を狙える（+22%）。球速も少し上がる

### サーブ

- 打つ前にパワーと回転量を設定する方式。**パワー・回転が強いほど狙ったコースに行きにくい**（散らばる）
- トスは常に統一トス（base約0.9m→頂点約3.35m→落下）。左右のコースはトス位置では決まらない（トス中もマウスで狙う場所を指定）
- 適正打点（ゲージの「適正」マーカー）で打つほど速く正確。**高すぎる打点は空振り**（フォルト）。ゲージは誤解を招く広い色付きゾーン（旧・緑/赤帯）を廃止し、適正点だけを細い1本線で示す
- サーブは**ボタン+Space（修飾キー）の組み合わせで4種**に分かれる（ラリー中の左=シュート/右=カット/Space+クリック=ロブと同じ考え方）。ゲージには4種すべての適正打点マーカーを表示

  | 操作 | 種類 | 特徴 |
  | --- | --- | --- |
  | 左クリック | フラット | 最速・最深。トス頂点付近の高い打点 |
  | 右クリック | スライス | フラットよりやや低い打点。フラットより遅れて落下する |
  | Space+左クリック | アンダーカット | 低いトス位置から山なり。セカンドサーブ向けの安全球で確実に入りやすい |
  | Space+右クリック | 攻撃的カット | フラットとアンダーカットの中間の打点で速く鋭く切れる/伸びる。適正ゾーンが狭くリスク高め |

- **全員（4人）の準備が整うまでサーブは打てない**: サーバー・両前衛・レシーバーが所定位置に到達し、レシーブ側が静止して初めてサーブが始まる（最大6秒で打ち切り）
- **相手サーブの種類は打つ前に表示**される（HUDとサーバー頭上のバッジ）。スライス/アンダーカットを見たら前へ詰める、という移動の時間が確保されている

### 相手AI（ソフトテニスのセオリー）

- **前衛の定位置（確定セオリー）**: 「相手後衛の打点 ─ 自コートのセンターマーク」を結んだ線上、ただし**気持ち一歩外側**（利き腕の肩が線に乗る程度）に立つ。相手後衛の前後の動きには**鏡のように対応**（歩幅の約半分の前後追従）。センターを打たせないことを最優先に、ストレートのレーンを担当する
- **後衛の定位置（確定セオリー）**: 前衛がストレートを守る前提で、後衛はそのストレートレーンを捨て、**残ったクロス側範囲の真ん中**（コート中央ではなくクロス寄り）にベースライン付近で立つ。これで前衛（ネット前＋ストレート）と後衛（後方＋クロス）の守備範囲が前後・左右ともに分離して被らない。ストレートへのロブは追わず、クロスへのロブは陣形が崩れるのでカバーに動く
- **前衛はレシーブが完了するまでポジション移動しない**（自チームのAI前衛も同様）。サーブ〜レシーブの間は定位置で待機し、レシーブが返ってからセオリー位置への移動・ポーチ判断を始める。打球ごとに ポーチに出る / ストレートを守る / ミドルを張る / 定位置 を選ぶ
- **相手後衛**には反応遅延・足の速さ・リーチの上限があり、前衛のいないコースへ良い打点から打てば抜ける。「前衛がコースを守り、後衛が走って拾う」という役割分担が成立する

### クロス展開／ストレート展開（陣形の動的切替）

ラリーがどちらの対角で行われているか（＝ボールがどのサイドへ進んでいるか）を判定し、自陣ペア・相手ペアの左右の立ち位置を展開に応じて動的に切り替えます。

- **クロス展開**（後衛同士が対角でラリー）: 「**後衛がいない方のサイドに前衛が立つ**」。自後衛が右なら前衛は左サイドのネット前。前衛はサイドへ寄りすぎてセンターを空けません
- **ストレート展開**（ボールがストレート＝同サイドへ展開）: **前衛と後衛が同じサイドに並ぶ（サイドバイサイド）**。前衛は「相手後衛の打点─自センター」線上でセンターより内側、後衛はストレート側ラインを担当します。クロスへロブが来ると後衛が大きく動かされ陣形が崩れるため、前衛が早めにクロス展開へ戻します
- 判定は「自陣後衛のいるサイド」と「相手の打球が向かうサイド」の一致/不一致で行い、小刻みな切替を防ぐヒステリシスを持たせています（`TUNING.pos` の `crossFrontX` / `straightFrontX` / `straightBackX` / `devHysteresis`）

### レシーブ順（確定セオリー）

- レシーバー2人は**1ゲームの間ずっと同じサービスコート（右/左）を受け持ちます**。サーブが右→左と交互に入るので、各ポイントのレシーバーは「そのサーブが入る側を担当する1人」です
- ゲームをまたぐ（サーブ権交代）と受け持ちを再設定します。担当でないもう1人は自陣の定位置で待機します
- レシーブ位置は対角サービスコートの真ん中で、アンダーカット告知なら前進、オーバー告知なら後退して構えます

## 操作方法

### PC（推奨・横画面）

横長のワイド画面でコートが中央に大きく表示され、スコア・球種・打球ボタンがその周囲に並びます。

確定操作（左手＝移動、マウス＝狙い・打つ）:

- **移動**: **WASD（左手）専用**。矢印キーは使いません。**打点ゾーン中も移動できます**（操作ロックなし）
- **狙い**: **マウス（右手）**。コート上でマウスを動かすと、マウスが指すコート地点へ着地点カーソル（狙う場所）が追従します。ストロークのため中もサーブ時も同じ。マウスがcanvas外に出たときは直前の狙いを保持します
- **打つ**: ボールが打点ゾーンに入ると**自動で「ため」が始まります**（長いほど鋭い角度）。**左クリック=シュート（フラット/ドライブ）、右クリック=カット（スライス/ドロップ）、Space（ロブ修飾キー）を押しながらクリック=ロブ**でその場でスイング。クリックした球種でそのまま打点へ打つ（マウス未操作はミドル深めへ）。ゾーン手前で早めにクリックしたときはゾーン到達時に同じ球種で自動スイング
- **球種**: 旧1/2/3キーの選択操作は廃止。シュートは打点高さで内部フラット/ドライブ、カットは**着地カーソルの深さ**で内部スライス/ドロップに自動振り分け
- **スマッシュ**: ネット前（前衛域）で高い球（ロブ等）が打点に来ると、球種選択に関わらず**自動でスマッシュ**（通常ストロークより速く、角度が鋭く下向きに突き刺さる決め球）になります
- **サーブ**: 画面のボタンでパワー・回転を設定 → **マウスで対角サービスコート内の狙う場所**を指す → 全員準備OK後に左クリックでトス（統一トス）→ ゲージの「適正」マーカー（4種それぞれの色）の高さで**左クリック=フラット、右クリック=スライス、Space+左クリック=アンダーカット、Space+右クリック=攻撃的カット**。立ち位置＋狙いで左/中/右を打ち分け、サービスコート外はフォルトです
- 画面右上の常設HUDに、サーブ中は4種の操作割当、ラリー中は左クリック/右クリック/Space+クリックの球種割当が常時表示されます

### スマホ（縦画面・簡易対応）

縦画面では横長コートを縮小して表示し、操作系は縦に積みます（当面は最適化していません）。

- **移動**: 左下のスティック
- **球種選択**: 画面下部に**3つのボタン（シュート/カット/ロブ）を常時表示**（系統ごとの色分け）。いつでもタップで即その系統を選択でき、ため中でも切り替えられる。アクティブな系統はハイライト表示
- **打つ**: 「打つ」ボタンをタップでスイング（打点ゾーンに入ると自動でためが始まる。カーソル未操作はミドル深めへ）
- **サーブ**: パワー・回転のボタンで設定 → 全員準備OK後に「トス / 打つ」ボタンでトス（統一トス）→ 「適正」マーカーの高さでもう一度タップ（フラットサーブとして打つ。スライス/アンダーカット/攻撃的カットはPCの右クリック・Space操作のみ対応）

## パラメータ調整

球速・足の速さ・打点品質の変換係数・AI制限（反応遅延/リーチ/ポーチ率）などは `app.js` 冒頭の `TUNING` オブジェクトに一元化しています。数値を書き換えるだけでゲームバランスを調整できます。
選手の能力値は `makeStats()`（power / serve / speed / reach / control / volley）に一元化しており、将来の育成要素はこの値を書き換えるだけで反映されます。
