# soft-tennis

PCとスマホで遊べるソフトテニス（軟式テニス）のダブルスラリーゲームです。
雁行陣（後衛＋前衛）の布陣で、相手前衛との駆け引きを楽しめます。

## 遊び方

1. 「試合を始める」ボタンを押して試合を開始
2. あなたは自チームの後衛を操作します。自チーム前衛と相手チーム（後衛・前衛）はAIが自動で動きます
3. 自分のサーブの番では、サーブ方式（アンダーカット／オーバー）を選んでから打ちます。サーブはベースライン後方からポイントごとに右／左交互に打ち、対角のサービスコートに入らなければフォルト（2本制、ダブルフォルトで失点）です
4. ラリー中は、コース（クロス／ミドル／ストレート）と球種（シュート／ロブ）を選んで打ち返します。リングのマーカーが着地予定地点を示し、自分側に来るボールは黄色で表示されます
5. 相手前衛がネット際でポジションを取り、低いボールが近くを通るとボレー・スマッシュでカットしてきます。前衛の位置を見て空いたコースを狙うか、ロブで頭上を越しましょう
6. ソフトテニスのポイント表記（0 → 1 → 2 → 3 → ゲーム）でスコアを表示。3-3はデュースとなり、2点差がつくまで続きます
7. 先に3ゲーム取った方が勝利（5ゲームマッチ）。サービスは1ゲームごとに両ペア4人の輪番で、2-2となった最終ゲームはファイナルゲーム（7ポイント先取・6-6デュース・2ポイントごとにサーブ交代）です

## ショットの仕組み

- **球種**：シュートボール（低く速い）／ロブ（高く弧を描き、前衛の頭上を越える）
- **コース**：クロス（対角線へ）／ミドル（中央）／ストレート（同サイドのライン際）
- **フォア・バック**：ボールが来た側によってフォアハンド／バックハンドが決まり、バック側は威力・精度が落ちます
- **サーブ**：アンダーカット（遅いがバウンド後に低く滑り、攻められにくい）／オーバー（速いがフォルトのリスクあり）

## 操作方法

- PC：← → キーで後衛を移動、A／S／Dでコース（クロス／ミドル／ストレート）選択、J／Kで球種（シュート／ロブ）選択、スペースキーで打つ
- スマホ：画面をドラッグして移動、画面下のボタンでコース・球種・サーブ方式を選択し、タップで打つ

## 技術構成

- Vanilla JavaScript（Canvas 2D、実コート寸法のメートル座標＋透視投影による3D風コート表現）
- 選手の能力値は `app.js` の `makeStats()`（power / serve / speed / reach / control / volley）に一元化しており、将来の育成要素はこの値を書き換えるだけで反映されます
- GitHub Pages（公開）
